# odoo-dev-helper
Helper scripts to ease odoo development

## The payslip lines test fixer
Example usage: `odoo-dev-helper/l10n_be_payslip_results_fixer.py --test-tags /test_l10n_be_hr_payroll_account:TestDMFA`
It will automatically run all tests that satisfy the test tags and modify the expected payslip lines results with the real ones.
